# RockPaperScissor
RockPaperScissor Mini Game
A simple and fun Rock Paper Scissors web game built using HTML, CSS, and JavaScript. Designed for interactivity and styled with responsive, beginner-friendly visuals.

**Technologies Used**
1. HTML5 – Structure and layout
2. CSS3 – Styling and hover effects
3. Vanilla JavaScript – Game logic and dynamic DOM interactions

**How to Play**
1. Open the index.html file in any modern browser.
2. Choose your move by clicking on Rock, Paper, or Scissors.
3. The computer will make a random choice.
4. The winner is displayed along with updated scores.
5. Click Reset Game to start fresh anytime.
